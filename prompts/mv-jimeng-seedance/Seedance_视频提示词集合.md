## 版本 001 - 仿生人趴着看电视

**日期**: 2026-03-03

### 需求描述

| 项目 | 内容 |
|------|------|
| **角色** | 仿生人（男），白衣服 |
| **姿态** | 趴着（沙发/床上），人物虚焦 |
| **焦点** | 电视屏幕（清晰对焦），人物虚焦 |
| **场景** | 昏暗房间，电视机闪烁照亮人物轮廓 |
| **视角** | 侧面或背面视角（非正面特写） |
| **运镜** | 固定镜头，一镜到底 |
| **风格** | 电影感，情绪片，黑白电影风格，浅景深 |
| **参考素材** | 1张黑白构图参考图 + 3张白衣服角色参考图 |

### 提示词方案

#### 方案 A：完整专业版

```
@参考图 作为画面构图与风格参考。

参考@角色参考图1 @角色参考图2 @角色参考图3 中的白衣仿生人男性，保持服装、妆造完全一致。

【主体】一个身穿白衣的仿生人男性，趴在沙发上，面向前方。

【动作】静静地注视着电视机屏幕，姿态放松，仿佛沉浸其中。

【场景】昏暗的房间，四周隐没在黑暗中，唯一光源来自前方的电视机。

【光影】电视机屏幕不断闪烁，白色光芒如呼吸般明暗交替，勾勒出人物的轮廓。焦点对准屏幕，人物处于景深之外呈现虚焦效果，形成浅景深的画面。

【镜头语言】固定机位，侧面/斜后方视角，一镜到底，不移动镜头。

【风格】黑白电影风格，情绪片，科幻悲剧感，孤独压抑氛围。

【画质】4K高清，高对比度，经典黑白摄影美学。

【约束】无文字叠加，无水印，画面稳定不抖动，焦点在电视屏幕，人物保持虚焦，黑白色调。
```

**日期**: 2026-03-03

### 需求描述

| 项目 | 内容 |
|------|------|

| **角色** | 男（黄色衣，仿生人）左侧 / 女（红色衣，最后人类）右侧 |

| **事件** | 烟花升空 → 炸开 |

### 提示词方案

#### 方案 A：完整详细版（推荐）

```
@参考图 作为尾帧风格参考

画面左侧，一个身穿黄色衣物的仿生人男性与右侧身穿红色衣物的女性紧紧相拥，两人背对镜头，面向远方地平线。

镜头初始时刻，两人几乎完全处于剪影状态，只有边缘轮廓被橙红色月光勾勒出微弱的轮廓光，人物细节隐没在黑暗中，营造出末世的孤独与凄美。

约3秒后，一颗烟花从画面后方地平线缓缓升起，拖着金色光尾划破夜空。

烟花升至画面中上部时炸开，绽放出巨大的金色与红色交织的火焰光芒，瞬间照亮两人的背影，人物轮廓从剪影转为半亮，衣褶与发丝在烟花光芒中清晰可见。

烟花绽放的光芒与橙红色月光交织，在沙丘上投下两人长长的影子，光影层次丰富，画面充满电影感与末世悲怆氛围。

```
