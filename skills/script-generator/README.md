# Script Generator Skill

This skill allows Claude to generate Xiaohongshu/YouTube scripts based on 4 pre-defined templates.

## Categories

1. **Tech Tweaks (干货技巧类)**
2. **Buying Guide (选购攻略类)**
3. **Product List (好物清单/推荐类)**
4. **Sponsored (单品推广/商单类)**

## How to Trigger

Simply ask Claude to generate a script or use keywords like "写个文案", "生成脚本", etc.

## Usage

- Provide the topic or product details.
- Claude will identify the category and use the appropriate template.
- If necessary information is missing, Claude will prompt you.

## Examples

- "帮我写一个iPhone省电技巧的脚本"
- "我想推荐几款平价平板电脑"
- "我要给石头扫地机器人写个推广文案"
