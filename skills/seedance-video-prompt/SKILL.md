## 核心概述

Seedance 2.0 是字节跳动推出的新一代多模态视频生成模型，支持**图片、视频、音频、文本**四种模态混合输入，单次最多可上传 **12 个文件**（9张图片 + 3段视频 + 3段音频），生成 **4-15 秒**视频。

### 核心能力
- 多模态参考：图片定风格、视频定运镜、音频定节奏
- 人物一致性：多镜头保持角色稳定
- 电影级运镜：自动规划专业分镜
- @ 语法：精确控制每个素材的用途

## 两种入口模式

### 模式 1：首尾帧模式（新手推荐）
**适用场景**：仅有一张首帧图或尾帧图 + 文字提示词

**操作流程**：
1. 上传一张图（定下画风）
2. 写一句描述
3. 生成视频

**经典案例**：
- 上传九宫格分镜图 → 一句提示词 → 完整视频生成
- 上传一张人物照 → 描述"这个女孩在雨中奔跑" → 视频成片

### 模式 2：全能参考模式（进阶必选）
**适用场景**：需要混合输入图片、视频、音频和文字

**核心优势**：
- 解锁导演级全部能力
- 精准复刻运镜、动作、氛围
- 支持视频延长、场景补全、音乐卡点

**何时选择**：
- 素材超过一张图
- 包含参考视频或音频
- 需要高度定制化效果

## @ 语法（核心功能）

### 为什么一定要用 @？
模型不会自动猜测你的意图。如果不明确告诉它每个素材的用途，它就可能乱用。

### @ 语法用法速查表

| 用法 | 示例 |
|------|------|
| **指定首帧** | `@图片1 作为首帧，定下整体画风` |
| **参考运镜** | `参考@视频1的运镜方式，从全景推到特写` |
| **指定角色** | `@图片1 中的女生作为主角` |
| **融合多视频** | `在@视频1和@视频2之间加一个场景转换` |
| **延长视频** | `将@视频1延长 5 秒` |
| **声音参考** | `@音频1 作为配乐，对齐画面节奏` |
| **负向提示** | `无文字叠加，无水印` |

### @ 语法实战案例

**场景**：制作品牌广告

**上传素材**：
- @产品图：产品照片
- @运镜视频：参考视频（展示相机运动方式）
- @配乐：15 秒背景音乐

**提示词**：
```
将@产品图放在画面中央，参考@运镜视频的旋转跟随方式，从俯拍推进到侧面特写，营造高级感。@配乐同步卡点，画面配色为冷色调，电影级色彩分级。
```

## 提示词黄金公式

### 完整公式（8要素）
```
[主体] + [动作] + [场景] + [光影] + [镜头语言] + [风格] + [画质] + [约束]
```

### 简化公式（S-A-C-S-C）
- **S**ubject（主体）+ **A**ction（动作）+ **C**amera（运镜）+ **S**tyle（风格）+ **C**onstraints（约束）

### 要素详解

| 要素 | 说明 | 示例 |
|------|------|------|
| **主体** | 角色、对象、主角 | 一个浑身缠绕金色雷电的少年剑士 |
| **动作** | 发生了什么 | 与恶鬼对峙、拔刀瞬间化作闪电 |
| **场景** | 发生在哪里 | 夜间古宅、暴雨中、血雾弥漫 |
| **光影** | 光线描述 | 黄昏暖光、侧光打亮、霓虹灯光 |
| **镜头语言** | 运镜方式 | 缓慢推镜头、360度环绕、俯拍视角 |
| **风格** | 整体视觉风格 | 日式热血动漫、电影级色彩、赛博朋克 |
| **画质** | 清晰度要求 | 4K高清、细节清晰、高对比度 |
| **约束** | 负向提示/限制 | 无文字叠加、稳定运镜、画面流畅 |

### 通用负向提示
```
无文字叠加，无水印，无logo，稳定运镜，画面流畅不抖动，细节清晰
```

### 按场景分类

| 场景 | 推荐负向提示 |
|------|-------------|
| 人物视频 | 无变形，无扭曲，五官端正，肢体自然 |
| 产品展示 | 无反光过曝，无色彩失真，背景干净 |
| 风景视频 | 无闪烁，无噪点，色彩还原真实 |
| 动作视频 | 无卡顿，动作连贯，无穿模 |

## 人物一致性方法（3层方案）

### 第1层：基础方案（最简单）
- 每次都上传同一张人物照作为首帧
- 在提示词中强调：`保持该人物的脸部和服装一致`

### 第2层：进阶方案（更精准）
1. 第一个视频生成后，截图中人物最好看的那一帧
2. 将截图作为后续视频的参考图
3. 用 @ 语法指定：`@参考图中的这个人物作为主角`

### 第3层：专业方案（95%+一致性）
1. 同时上传 3-5 张人物的不同角度照
2. 在提示词中清晰描述人物特征：
   ```
   长直黑发、眼睛大、肤色白、穿黑色连衣裙的女生，
   @图片1作为正面参考，@图片2作为侧面参考
   ```
3. 每次生成都引用这些参考图

## 九宫格分镜法（大神技巧）

### 什么是九宫格分镜？
九宫格是一个 3x3 的图表，代表视频的 9 个关键帧。

### 操作步骤
1. 用 Figma/Canva/Procreate 画一个 9 宫格
2. 在每个格子里勾勒出关键画面（简笔画即可）
3. 上传九宫格图 + 一句提示词 → 完整视频

### 效果
- 提高一致性 50%+
- 减少"抽卡"次数
- 更精确控制视频节奏

### 示例提示词
```
@九宫格分镜图 按照从左到右、从上到下的顺序演绎，
保持人物一致，动作流畅连贯，4K画质
```

## 音乐卡点

### 基础用法
上传音乐文件后，提示词中加入：
```
@音乐1 中 X秒处的鼓点，同步主角的转身动作

### 进阶用法
```
0-2秒：静谧古琴，建立场景
2-5秒：逐渐加入鼓点，情感升温
5-10秒：高潮鼓声，激烈动作

## 常见问题解答

### Q1：为什么人物"换脸"了？
**原因**：没有明确指定参考图，或参考图不够清晰

**解决方案**：
1. 上传高清参考图（最少 400x400px）
2. 在提示词中加入：`@参考图中的这个人物作为主角，保持脸部特征完全一致`
3. 避免要求过于剧烈的表情变化

### Q2：镜头运动太生硬？
**原因**：提示词中缺乏相机运动描述

**解决方案**：
```
添加镜头描述：
- 镜头从全景缓慢推进到特写
- 摄像机绕着主角360度旋转
- 俯拍视角，自上而下观察
- 跟随镜头，与主角同步移动
- 稳定运镜，画面流畅不抖动
```

### Q3：生成太贵？省钱技巧
**建议方案**：
1. **迭代时用短时长**：先生成 4-5 秒看效果，满意后再生成 10-15 秒
2. **一次生成多个变体**：选最好的，避免频繁重试
3. **建立提示词库**：复用优质模板，减少试错次数
4. **用模板迭代**：从本文模板直接改参数，成功率 80%+

### Q4：多个视频如何保持人物一致？
**3层方案**：参见上方"人物一致性方法"章节

### Q5：音乐与画面不同步？
**解决方案**：
1. 上传参考音乐文件
2. 提示词中写：`@音乐1 作为背景音乐，卡点对齐`
3. 或精确描述：`0-2秒静谧，2-5秒逐渐加强，5-10秒高潮`

## 使用技巧总结

### 核心原则
1. **具体描述**：用具体的词语代替抽象描述
   - 好："一位穿白色连衣裙的女孩在海边奔跑"
   - 差："一个人在运动"

2. **添加细节**：补充光线、色彩、质感等细节
   - "金色阳光"、"柔和的阴影"、"丝绸般的质感"

3. **明确运镜**：指定摄像机的运动方式会让视频更有电影感

4. **风格统一**：保持风格和氛围的一致性

5. **合理长度**：提示词建议 80-200 字效果最佳

6. **必写约束**：动作、镜头、画质、约束这4个维度是必写的

### 新手3步快速上手

**Step 1**：体验首尾帧模式
- 找一张喜欢的图
- 写一句简单描述
- 看看效果

**Step 2**：试试全能参考模式
- 上传 1 张图 + 1 段参考视频
- 用 @ 语法指定素材用途
- 对比效果差异

## 工作流程

### 当用户请求生成视频提示词时：

1. **了解需求**
   - 询问用户想创作什么类型的视频
   - 确认是否有参考素材（图片/视频/音频）
   - 了解偏好风格和氛围

2. **选择模式**
   - 新手/简单需求 → 首尾帧模式
   - 进阶/多素材 → 全能参考模式

3. **构建提示词**
   - 使用8要素公式
   - 添加必要的负向提示
   - 如有素材，使用@语法

4. **提供多个版本**
   - 提供2-3个不同风格的版本
   - 标注每个版本的特点

5. **后续优化**
   - 根据用户反馈调整
   - 提供人物一致性建议
   - 推荐相关模板
