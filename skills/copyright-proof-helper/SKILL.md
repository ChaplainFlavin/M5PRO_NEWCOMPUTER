# Copyright Proof Helper

Use this skill when the user needs to produce or revise copyright-related supporting documents for delivered creative work.

## Workflow

1. Identify the document type:
   - authorization letter
   - copyright proof statement
   - supplementary explanation
   - screenshot/self-made visual risk clarification
2. Reuse the closest reference text from `references/`.
3. Preserve the user's company name, project title, counterparty name, and risk-allocation intent.
4. If the user mentions self-made screenshots, mock pages, ad visuals, UI screens, or simulated popups, explicitly cover:
   - the visuals were independently planned/designed/produced for the project
   - they are original or simulated content for expression only
   - they do not unlawfully copy third-party protected works
   - any resulting IP liability is borne by the producing company
5. Keep the wording client-facing and review-friendly. Strengthen liability and indemnity clauses when the user wants a “偏客户口径” version.
