# 6m×6m 商业摄影棚设计与施工方案

**项目信息**
- 面积：6m×6m（36平米）
- 层高：4米以上
- 状态：毛坯房
- 用途：中型商品拍摄（服装、包袋、鞋类）
- 需求：产品拍摄、模特拍摄、视频拍摄
- 预算：设备齐全，仅需施工费用

**创建日期：** 2025-01-28
**版本：** 1.0

---

   - 充分保证机位退让空间（最深可退缩近5米）。

   - 落地式四轴背景纸架（可容纳 2.7m 标准卷）。

   - **底层北侧（通道区）**：约留 1 米缓冲地带。安置货架侧面的攀爬滑轮挂梯（梯下可放饮水机或花架）。
   - **货架高层（2.2m - 3.5m）**：主要储物空域。存放设备器材箱（桌面上空）、背景道具备件（更衣室上空）。

## 二、墙面处理方案

### 材料选择

**北墙（背景墙）：**
- 深灰色哑光乳胶漆
- 或黑色吸音布帘（可收放）
- 目的：减少反光，创造可控光环境

**其他三面墙：**
- 中性灰（18%灰）或白色哑光漆
- 东、西墙可加装波浪吸音板
- 目的：减少回声，适合视频收音

**有窗墙面（东墙、南墙）：**
- 安装全遮光卷帘（黑色）
- 窗帘结构：
  - 外层：加厚遮光布
  - 内层：黑色绒布（完全阻挡光线）
- 窗户下方可做矮柜（收纳道具）

**墙面保护：**
- 1.2米高墙裙（深色毡布或软包）
- 防止灯光架刮墙
- 预留挂钩安装点

### 材料清单

| 材料 | 数量 | 用途 |
|------|------|------|
| 深灰色哑光乳胶漆 | 5-10升 | 北墙 |
| 中性灰/白色乳胶漆 | 10-15升 | 其他三面墙 |
| 抗碱底漆 | 5升 | 墙面封闭 |
| 腻子粉 | 2-3袋 | 墙面找平 |
| 遮光窗帘布 | 按窗户尺寸 | 双层窗帘 |
| 吸音板（可选） | 20-30平米 | 西墙、东墙 |
| 墙裙毡布 | 约30平米 | 墙面保护 |

## 三、地面处理方案

### 推荐方案：灰色自流平+地坪漆

**优点：**
- 无缝、易清洁
- 反光适中
- 耐用、专业

**施工步骤：**

1. **基础处理**
   - 原地面清理
   - 修补裂缝、坑洞
   - 涂刷界面剂

2. **自流平施工**
   - 搅拌自流平水泥
   - 倒在地面上刮平
   - 放气泡
   - 养护24小时（禁止踩踏）

3. **地坪漆施工**
   - 第一遍底漆（滚涂）
   - 干燥4-6小时
   - 第二遍面漆（灰色哑光）
   - 干燥12-24小时
   - 第三遍面漆（如需要）
   - 完全固化48小时后使用

**颜色选择：**
- 中灰色（减少地面反光干扰）
- 哑光处理（避免反光）

### 材料清单

| 材料 | 数量 | 用途 |
|------|------|------|
| 自流平水泥 | 50-60平米 | 地面找平 |
| 界面剂 | 1-2桶 | 增加粘结力 |
| 环氧地坪漆 | 50-60平米 | 面层 |
| 稀释剂 | 适量 | 调节漆的粘度 |

## 四、电路布置方案

### 电力需求

- LED灯光：2000-3000W
- 电脑/显示器：500-800W
- 空调/通风：2000-3000W
- 其他设备：500W
- **总功率：约5-7kW**

### 电路设计

**主进线：**
- 建议380V三相电，或220V单相40A以上
- 独立电箱

**回路分配：**

1. **灯光回路**（32A）
   - 四周墙壁插座（每面墙3个，距地30cm）
   - 选用工业级16A三孔插座

2. **设备/电脑回路**（20A）
   - 工作台区域
   - 避免与灯光同回路

3. **空调/通风回路**（25A）
   - 空调专用

**空开配置：**
- 总开：63A × 1
- 灯光回路：32A × 1

## 八、施工顺序

### 总时间：21个工作日

```
┌─────────┬────────────────────────────────────┐
│ 阶段     │ 工作内容                           │
├─────────┼────────────────────────────────────┤
│ 第1-2天  │ 前期准备、材料采购                 │
│ 第3-7天  │ 电路改造（开槽、布管、穿线、测试） │
│ 第8-12天 │ 墙面处理（基层、腻子、刷漆）       │
│ 第13-17天│ 地面处理（自流平、地坪漆）         │
│ 第18天   │ 窗户遮光处理                       │
│ 第19-20天│ 空调、通风安装                     │
│ 第21天   │ 设备进场、测试验收                 │
└─────────┴────────────────────────────────────┘
```

### 关键原则

**施工顺序：**
1. 隐蔽工程先做（电路）
2. 由上到下（先墙面后地面）
3. 由里到外（先装修后设备）
4. 先脏后净（先粉尘作业后精细作业）

**❌ 错误顺序：**
- 先刷漆再开槽布线 → 破坏墙面
- 先做地面再刷漆 → 漆滴落污染地面

## 十一、施工注意事项

### 安全要点

1. **电路工程**
   - 必须由持证电工操作
   - 完成后必须做绝缘测试
   - 拍照留底（线路位置）

2. **顶部作业**
   - 4米层高需要脚手架
   - 戴安全帽
   - 系安全绳

3. **地面施工**
   - 自流平施工穿钉鞋
   - 养护期间禁止踩踏

### 质量控制

1. **电路测试**
   - 绝缘电阻测试
   - 漏电保护测试
   - 满负荷运行测试

2. **墙面验收**
   - 平整度检查
   - 色差检查
   - 无流挂、刷痕

3. **地面验收**
   - 平整度误差≤2mm
   - 无气泡、开裂
   - 固化完全

4. **遮光测试**
   - 完全黑暗
   - 无漏光点

### 避坑提醒

**电路：**
- ✅ 预留足够的插座和容量
- ❌ 不要省电线（2.5平方起）
- ✅ 分回路控制
- ❌ 不要把所有设备接在一个回路

**墙面：**
- ✅ 先刷底漆
- ✅ 两遍面漆
- ❌ 不要阴雨天刷漆

**地面：**
- ✅ 自流平要充分养护
- ✅ 地坪漆要完全固化
- ❌ 不要急于使用

**照明：**
- ✅ 使用标准接口（E27）
- ✅ 安装高度2.8米
- ❌ 不要用特殊接口灯泡

## 十二、测试验收标准

### 遮光测试

- 关闭所有灯光
- 房间完全黑暗
- 无光线透过窗帘

### 电力测试

- 所有灯光同时开启（满负荷）
- 电压稳定
- 无跳闸、发热

### 拍摄测试

- 使用标准色卡
- 检查墙面反光
- 检查地面反光
- 测试不同布光方案

### 视频收音测试

- 录制测试视频
- 检查回声
- 检查空调噪音影响

### 安全测试

- 灯架稳定性
- 背景纸架稳固性
- 线缆走线安全

## 十三、日常使用建议

### 灯光控制

**日常模式：**
- 开启壁灯（全开或分区）
- 关闭摄影灯

**拍摄模式：**
- 关闭壁灯
- 开启摄影灯
- 关闭空调/通风（收音时）

### 维护保养

1. **定期检查**
   - 每月检查电路
   - 每季度检查墙面
   - 每年检查地面

2. **清洁保养**
   - 墙面：用干布擦拭
   - 地面：用中性清洁剂
   - 窗帘：定期吸尘

3. **设备收纳**
   - 用完归位
   - 避免堆积
   - 保持通道畅通
