# 顺德古朗文旅推广方案 - 策划面板

## 📝 核心目标与初始约束
- **项目目标**: 撰写一份针对顺德古朗村的**文旅规划案**（重点偏向实体规划、文旅业态与长效体验，而不仅是线上的传播）。
- **前置设定/参考基准**: 用户提供了一份《临潼三农传播案》作为风格和套路参考。
  - **参考案拆解（阿康视角）**: 临潼案的骨架是标准的单平台（抖音）漏斗模型。
- **项目区隔与致命约束（16日最新沟通）**: 
  1. **受众降维，拒绝假大空**：汇报对象是村委会领导。必须极其“接地气”，算清投入产出比。
  2. **兼顾现状，拒绝盲目复刻**：村里想学隔壁已成气候的“逢简水乡”（咖啡/甜品/民宿聚集），但古朗必须走**极致的差异化**。古朗停车难、无大型步道商业街，只能做“**静村**”。
  3. **生意引擎先行（文创做切口）**：抛弃同质化的咖啡车，切入点是“**卖体验与小文创创收**”，以点带面。
  4. **痛点攻坚（阿康新补充）**：
     - **选品需扩充**：长寿果受季节限制，需补充更丰富的非时令文创品类。
     - **动线需“无中生有”**：利用“古朗文化广场”作起终点（解决停车），以“志愿服务站”为补给中枢。完美复刻“熊野古道”模式：**设立低维护成本的特色打卡集章柱（24小时营业）+ 沿途自助无人售货柜（测试人性与高转化率）**。
     - **人群画像纠偏（核心定调）**：抛弃烂大街的“反内卷/班味”网红化标签（容易引起审美疲劳，且实地如财神庙等基建较破败承接不住）。**核心主力客群锁定为【珠三角周末家庭游（拖家带口团队）】**。
     - **产品线迭代**：保留“盲盒”和“集章”机制，立意完全从“青年发泄”转向“家庭互动/长者祈福/亲子研学/家族寻根”。
     - **年案思维（传播线补充）**：需补充全年的节点营销图谱（按四季/传统节日布局活动）。
- **古朗村核心 Tag & 战略爆破点（阿康共识）**: 
  - 核心资源：“百岁坊”（长寿）、“节孝坊”、丰富的宗祠群（保留15座）、**极具分量的“御赐牌匾”**、原生态水乡（更安静、偏养老）。这种“隐秘沉静的高级感”是核心护城河。
  - **核心 Concept（大创意）**：**东方水乡版的“熊野古道/朝圣之路”**。利用现有的弯绕小路和路牌，打造一条“长寿/祈福静心徒步线”。
  - **核心盈利产品**：**通关文牒/祈福集章册**。游客付费购买底册，通过在各宗祠、古桥盖章收集，最终兑换高价值的在地文创周边。这套玩法老一辈和徒步青年都极其吃香，且对村庄物理环境零破坏、零基建要求。
