# 腾讯会议 远程参会操作指南

> 本指南帮助您在 **5 分钟内** 完成腾讯会议的安装、调试与入会，全程配图说明。

---

## 一、您需要准备的东西

| 准备事项 | 说明 |
|----------|------|
| **一台电脑** | 笔记本电脑最佳（自带摄像头和麦克风），Windows 10 以上系统 |
| **一副耳机** | 插上耳机可以彻底避免回音，普通手机耳机即可 |
| **稳定的网络** | 连接 Wi-Fi 或插网线，**提前关掉正在下载的东西** |
| **安静的环境** | 关闭门窗，面部朝向光源（脸要亮，背后不要逆光） |
| **插好充电器** | 全程保持充电，防止中途断电 |

---

## 二、安装腾讯会议

### 第 1 步：下载

打开电脑浏览器，在地址栏输入以下网址并回车：

```
https://meeting.tencent.com/download/
```

网页打开后，点击页面上的 **【免费下载】** 按钮。

> 🖼️ **截图①：** 截取腾讯会议官网下载页面 → 红框圈住 **"免费下载"按钮**

---

### 第 2 步：安装

1. 下载完成后，找到刚下载的文件（通常在屏幕左下角或"下载"文件夹），**双击打开**
2. 如果电脑弹出提示"是否允许此应用进行更改"，点 **【是】**
3. 勾选 ☑ **"已阅读并同意"**，然后点 **【安装】**
4. 等待安装完成，点 **【完成】** 即可

> 🖼️ **截图②：** 截取安装界面 → 红框圈住 **"已阅读并同意"勾选框** 和 **"安装"按钮**

---

### 第 3 步：登录

安装结束后腾讯会议会自动打开，出现登录界面：

- 推荐点击 **【微信登录】**，用手机微信扫一下屏幕上的二维码即可登入
- 也可以选择【手机号登录】，输入手机号和短信验证码

> 🖼️ **截图③：** 截取登录页面 → 红框圈住 **"微信登录"按钮** 和 **微信扫码区域**

---

## 三、入会前：检查您的摄像头和麦克风

> ⚠️ **请在正式会议前一天完成此检查，确保设备正常。**

### 第 4 步：打开设置

登录后进入腾讯会议主界面，点击左下角的 **【设置】**（齿轮图标）。

> 🖼️ **截图④：** 截取腾讯会议主界面 → 红框圈住左下角 **"设置"齿轮图标**

---

### 第 5 步：测试声音

1. 在设置页面，点击左侧的 **【音频】**
2. 找到 **"扬声器"** 一栏，点击 **【检测】**
   - ✅ 听到声音 → 正常
   - ❌ 没有声音 → 检查耳机是否插好，或换一个耳机试试
3. 找到 **"麦克风"** 一栏，点击 **【检测】**
   - ✅ 对着电脑说话，看到绿色音量条在跳动 → 正常
   - ❌ 音量条不动 → 检查耳机上的麦克风开关，或检查电脑是否禁用了麦克风

> 🖼️ **截图⑤：** 截取音频设置页面 → 红框分别圈住 **"扬声器-检测"按钮** 和 **"麦克风-检测"按钮**，用箭头指向绿色音量条

---

### 第 6 步：测试摄像头

1. 在设置页面，点击左侧的 **【视频】**
2. 预览区域应出现您的画面
   - ✅ 能看到自己 → 正常
   - ❌ 黑屏或无画面 → 检查笔记本摄像头是否有物理遮挡开关（部分笔记本有一个小滑块）
3. 如果想要画面更好看，可以打开下方的 **【美颜】** 开关

> 🖼️ **截图⑥：** 截取视频设置页面 → 红框圈住 **摄像头预览画面** 和 **"美颜"开关**

---

## 四、加入会议

### 方式 A：点击链接入会（最简单 ⭐ 推荐）

会前，工作人员会通过微信发给您一个 **会议链接**。

1. 在 **电脑微信** 上点击该链接
2. 电脑会自动弹出腾讯会议窗口
3. 确认以下两项已勾选：
   - ✅ 入会开启麦克风
   - ✅ 入会开启摄像头
4. 点击 **【加入会议】**

> 🖼️ **截图⑦：** 截取入会确认弹窗 → 红框圈住 **"入会开启麦克风"** 和 **"入会开启摄像头"** 两个勾选项，以及底部 **"加入会议"按钮**

---

### 方式 B：手动输入会议号入会（备用）

如果链接打不开，可以手动操作：

1. 打开腾讯会议，在主界面点击 **【加入会议】**
2. 输入工作人员发给您的 **9 位数会议号**
3. 在"您的名称"处填入您的 **姓名**
4. 勾选"入会开启摄像头"和"入会开启麦克风"
5. 点击 **【加入会议】**

> 🖼️ **截图⑧：** 截取手动入会页面 → 红框圈住 **"会议号输入框"** 和 **"加入会议"按钮**

---

## 五、进入会议后

### 您只需要认识底部这 3 个按钮

进入会议后，屏幕 **最下方** 有一排按钮：

| 按钮 | 样子 | 怎么用 |
|------|------|--------|
| **静音 / 解除静音** | 🎤 麦克风图标 | 点一下切换开关。**轮到您发言时，确保显示"解除静音"状态（图标无红色斜杠）** |
| **开启 / 关闭视频** | 📷 摄像头图标 | 全程保持开启即可 |
| **共享屏幕** | 🖥️ 显示器图标 | 播放 PPT 课件时使用（见下方说明） |

> 🖼️ **截图⑨：** 截取会议中的底部工具栏 → 红框分别圈住 **"静音""开启视频""共享屏幕"** 三个按钮，旁边标注文字说明

---

### 播放 PPT 课件的操作步骤

如果您需要在会议中展示 PPT：

1. **会前**：提前用 WPS 或 PowerPoint 打开您的课件文件
2. **会中**：点击底部的 **【共享屏幕】**
3. 在弹出的窗口中，选择 **您打开课件的那个窗口**
4. 点击 **【确认共享】**
5. **翻页**：只需按键盘上的 **空格键**，每按一次翻一页
6. **讲完后**：点击屏幕顶部出现的 **【停止共享】**

> 🖼️ **截图⑩：** 截取共享屏幕的窗口选择界面 → 红框圈住 **课件窗口** 和 **"确认共享"按钮**

> 💡 **小贴士：** 如果中间课件画面消失或按错了，**不用紧张，继续对着摄像头讲即可**。现场工作人员会随时帮您处理，您的声音和画面全程都在。

## 六、常见问题怎么办

| 遇到的情况 | 怎么解决 |
|------------|----------|
| 对方听不到我说话 | 看看底部麦克风按钮是不是 **红色斜杠状态（静音中）**，点一下取消静音 |
| 我听不到对方说话 | 检查耳机是否插好；检查电脑右下角小喇叭音量是否调到最低 |
| 摄像头画面是黑的 | 检查笔记本摄像头滑块是否打开；检查是否有其他软件占用了摄像头 |
| 画面很卡、很模糊 | 关掉其他正在看的视频或网页；靠近路由器或改用网线 |
| 回音很大 | **戴上耳机** 即可解决 |
| PPT 翻不了页 | 用鼠标点一下 PPT 画面（让它获得焦点），然后再按空格键 |
