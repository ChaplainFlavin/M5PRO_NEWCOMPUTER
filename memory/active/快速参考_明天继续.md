## 🎯 今日新增功能

### 1. 上下文记忆 + 连续调整
```
用户: 我是Bee，帮我写个TWS耳机推荐
→ 生成 + 保存上下文

用户: 字少点，500字
→ 读取上下文 → 重新生成500字

用户: 更吐槽一点
→ 读取上下文 → 保持500字，更吐槽风格
```

### 2. 结构化需求提取
```
我是Bee，帮我写一个商单，关于DJI ACTION6，
核心卖点是稳定性、大CMOS、暗光画质，参数请上网查
→ 提取：类型=商单，卖点=3个，要求=查参数
```

### 3. 口播稿转换
```
[生成文案后]
用户: 转口播稿
→ 带时间轴、场景标记、画面提示

[或直接粘贴文案]
自动识别 → 转换
```

```

```
