# Memory Domain: Workflow Prompting (工作流提示词法典)

*适用场景：执行文本到图像/视频的工作流，撰写正反向提示词时必须开局靶向调取的字典。*

### 1. 国内模型（即梦、可灵等）的审核求生规则
国内模型对涉嫌暴力、血腥、过度露骨的词汇极其敏感，会直接中断生成（红温拦截）。
* **颜色规避：** 凡是带“血”的颜色感觉，全部替换为**“暗赤色、朱红色”**。切勿盲跑“深红色、血红色”。
* **武器规避：** 避免使用“锋利长枪、刀剑”，一律安全替换为**“古典长柄神兵、抽象神话法器”**。
* **对抗词规避：** 避免在系统级提示词中使用“咒语、必须、强制、缺失”等命令型甚至反抗型语汇，必须转译为温和的设计客套话，如**“设定参考、排版展示、需要视觉确保”**。
* **英文排雷：** 当心模型因为解析大字负面英文词（如 `NO ARMOR`、`NO FIRE`）发生崩溃反噬，必须转换为自然散文短语，如 `without any armor`。

### 2. 即梦 (Dreamina) 脱离垫图的画风统一定轨
在即梦这类欠缺精准“风格提取（Style Reference UI）”的工具中同化统一多组镜头及 IP 的人物画风，必须抛弃垫图碰运气玄学，采用**“强力渲染环境描述句的前后缀夹击锁定”法**。
* **2.5D 微渲染手办风定调文本墙（可一键替换掉廉价水墨风的终极解）**：
  * **前缀必带起手式：** `(顶级大师杰作，极高画质，8k分辨率，次世代游戏引擎渲染:1.4)。Unreal Engine 5 (虚幻5引擎) 微渲染效果，2.5D精美手办质感，Octane Render (OC渲染器) 电影级打光。`
  * **后缀必带防崩式：** `极致细腻的PBR材质结构，全局光照 (Global Illumination)，(角色带有精致的树脂/泥塑质感外边缘，立体阴影强烈:1.2)。非扁平2D，非完全写实真人，绝美元模微距摄影感。`
  * **强力负面环境排除流：** 即使写在正面框也有效的规避组合 `扁平的，传统的2D水墨画，素描，完全逼真的照片，模糊，劣质3D，塑料感强，多余的环境，变形，低分辨率`。
